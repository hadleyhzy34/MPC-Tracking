# mpc_python_traj
This repository is an implementation of a pyhton/ros/casadi based mpc trajectory tracking application.

## Environments

- python 2.7(python 3.x should also work)
- Casadi 3.5.5
- numpy
- matplotlib
- bisect
- rospkg

## Installation

```
pip install -r requirements.txt
```
